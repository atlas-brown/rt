# Dataset Card for InterCode-ALFA-Data

<!-- Provide a quick summary of the dataset. -->
This is a manually corrected version of the InterCode-Bash dataset, provding natural langauge prompts and Bash commands for the task of machine translation.
This dataset is used by the InterCode-ALFA benchmark to assess the capabilities of NL2Bash translation models.
[InterCode-ALFA Benchmark](https://pypi.org/project/icalfa/)
